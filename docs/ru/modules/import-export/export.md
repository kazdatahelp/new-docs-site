### Основной Поиск
- Название продукта
- Код ТН ВЭД
- Страна назначения
- Период времени
- Диапазон стоимости

### Расширенные Фильтры
- Несколько стран
- Группы продуктов
- Пользовательские периоды
- Пороговые значения
- Индикаторы роста
