# Поиск Продуктов

Узнайте, как искать и анализировать продукты в международной торговле, используя модуль Импорт/Экспорт KazDATA.

## Функции Поиска

### Основной Поиск
- Название продукта
- Код ТН ВЭД
- Категория
- Описание
- Ключевые слова

### Расширенные Фильтры
- Несколько продуктов
- Группы продуктов
- Пользовательские категории
- Диапазоны стоимости
- Объем торговли

- Коды (8 цифр)
